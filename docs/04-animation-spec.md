# 气球动效说明

## 动效目标
通过气球、绑线和小剪刀交互，让用户明确感知“从任务入口进入工作流”的过程，形成有记忆点的工作台体验。

## 默认状态
- 六个气球在工作台中缓慢悬浮
- 每个气球通过绑线连接到中央聊天入口
- 绑线需要清晰可见，但不过度抢视觉
- 中央聊天入口处于待触发状态

## 悬停状态
- 鼠标移入气球时，光标切换为小剪刀
- 鼠标移入绑线时，光标同样切换为小剪刀
- 当前气球可以略微上浮或轻微放大
- 当前绑线可以轻微绷紧或提亮
- 中央聊天入口可出现弱高亮响应，提示即将进入工作流

## 点击状态
- 点击气球或绑线后，先出现剪断反馈
- 绑线应有明确断开动作
- 被触发的气球沿上方方向漂浮离场
- 中央聊天入口立即高亮
- 随后弹出对应工作流弹层

## 节奏建议
- 悬停反馈要快，建议在短时间内完成
- 剪断动作要利落，但不要突兀
- 气球上浮要轻盈顺滑
- 弹层出现要承接上一个动作，避免生硬切换

## 重置机制
- 需要提供一键重置入口
- 重置后恢复气球位置、绑线连接和聊天入口状态
- 重置适用于原型演示反复测试

## 动效气质
- 专业
- 轻柔
- 清晰
- 不夸张
- 不卡通化
