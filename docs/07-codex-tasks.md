# 开发任务与限制

## 开发任务
1. 搭建 Vite + React + CSS 的基础结构。
2. 完成左侧可折叠导航和页面骨架。
3. 构建 Workbench 页面，放置六个任务气球和中央聊天入口。
4. 实现气球与聊天入口之间的绑线展示。
5. 实现气球和绑线悬停时的小剪刀反馈。
6. 实现点击后剪断绑线、气球上浮、聊天入口高亮和弹层启动。
7. 完成工作流步骤：上下文构建、Prompt 预览、输出质检、版本优化。
8. 完成 Task Library、My Workflows、Results Library、Skill Records、Membership 页面。
9. 使用本地数据文件管理任务、工作流和方法记录内容。
10. 提供工作台重置能力，便于原型演示。

## 文档与实现要求
- 保持产品定位一致
- 保留 Skill 概念，但仅用于方法标签、提示、记录和复盘
- 使用语义化类名
- 保持组件可复用、可读
- 相同任务内容不要重复硬编码

## 禁止事项
- 不要把产品做成课程 App
- 不要把产品做成普通聊天工具
- 不要移除左侧导航
- 不要取消左侧导航折叠能力
- 不要移除气球、绑线、聊天入口或小剪刀动效
- 不要把任务主入口替换成普通卡片网格
- 不要引入任何游戏化成长表达
