# 视觉规范

## 整体方向
- 干净、专业、克制的 AI 工作台
- 有留白感，但不空
- 有轻盈感，但不幼稚
- 强调方法感与任务感，而不是娱乐感

## 颜色建议
- 主背景：偏暖的浅灰白或低饱和中性色
- 侧边导航：略深一层的中性色块，保证结构清晰
- 主强调色：冷静的蓝绿色或雾感蓝色
- 辅助色：用于区分不同任务气球，可采用低饱和粉、蓝、绿、橙
- 提示色：输出质检与提醒使用偏柔和的琥珀色或珊瑚色

## 配色原则
- 不做大面积炫彩渐变
- 不使用过于儿童化的高饱和配色
- 保持文本与背景有足够对比
- 气球颜色可以区分任务，但整体仍需统一

## 布局规范
- 目标画布固定为 1440 x 1024
- 左侧导航固定存在，支持折叠
- 主区域以工作台为视觉中心
- 中央聊天入口要成为动线焦点
- 气球分布要有呼吸感，避免挤成一团

## 组件气质
- 导航像专业工具，而不是学习平台
- 气球像轻量任务入口，而不是玩具元素
- 标签要简洁克制，强调方法信息
- 弹层与卡片遵循统一圆角、阴影和间距系统

## 参考方向
- Notion 风格的信息秩序感
- Linear 风格的专业与克制
- 一点点轻盈的实验性交互表达

## 明确限制
- 不是课程产品视觉
- 不是普通聊天产品视觉
- 不去掉左侧导航
- 不取消气球、绑线、聊天入口和小剪刀交互
