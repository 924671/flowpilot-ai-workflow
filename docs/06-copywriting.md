## 工作流步骤文案
- Context Builder：Add your goal, audience, source material, and constraints.
- Prompt Preview：Review the structured prompt before generating.
- Output Check：Check whether the result is clear, complete, and supported.
- Version Optimization：Create alternative versions for different work scenarios.
