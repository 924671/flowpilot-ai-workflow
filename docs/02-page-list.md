# 页面清单

## 1. Workbench
作用：
作为产品主入口，以六个悬浮气球承载六类高频任务。用户通过剪断绑线进入对应工作流。

页面内容：
- 左侧可折叠导航
- 六个任务气球
- 气球与聊天入口之间的绑线
- 中央聊天入口
- Skill 标签
- 重置入口

## 2. Task Library
作用：
集中展示所有任务卡片，方便用户按场景浏览和选择任务。

页面内容：
- 任务名称与简介
- 适用场景
- Skill 标签
- 进入工作流的入口

## 3. My Workflows
作用：
查看已经开始、进行中或完成过的工作流记录。

页面内容：
- 工作流卡片
- 最近编辑时间
- 当前进度说明
- 再次打开入口

## 4. Results Library
作用：
沉淀历史输出结果，方便回看、对比和继续优化。

页面内容：
- 结果列表
- 版本说明
- 输出质检备注
- 再利用入口

## 5. Skill Records
作用：
记录用户在真实任务中用过的方法，并做复盘整理。

页面内容：
- Common Skills
- Recent usage
- Common issues
- Reusable suggestions

示例内容：
- Context Expression：常用于项目复盘、周报、运营方案
- Output Check：常发现数据支撑不足、结论模糊、下一步不明确
- Version Optimization：可把一版输出转成汇报版、同步版、提纲版

## 6. Membership
作用：
展示会员方案、功能范围和升级信息。

页面内容：
- 方案说明
- 功能差异
- 升级入口
