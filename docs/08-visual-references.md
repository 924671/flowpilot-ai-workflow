# 08 Visual References

## 使用说明

本文件只用于整理 FlowPilot 的视觉找图方向，不涉及页面实现，不作为直接临摹清单。

整理原则：
- 参考的是信息气质、布局关系、交互氛围与视觉语言，不是照抄具体界面。
- 所有参考都应服务于 FlowPilot 作为 AI 工作流工具的定位，而不是课程产品、游戏化产品或普通聊天工具。
- 图片后续请人工筛选保存到对应文件夹，本次不下载任何图片。

---

## 1. Chat Entry

- 参考用途：寻找 Workbench 中央聊天入口的视觉表达方式，重点看“任务启动中枢”而不是普通对话框。
- 推荐搜索关键词：
  - `AI workbench central input desktop`
  - `command center ui desktop`
  - `floating input panel productivity app`
  - `assistant entry card interface`
  - `task launch input module`
- 适合借鉴的点：
  - 中央入口作为视觉焦点的权重控制
  - 输入区与状态提示的层级关系
  - 轻量高亮、边框、阴影带来的可触发感
  - 入口与周边留白之间的平衡
  - “开始任务”而非“闲聊对话”的语义氛围
- 不要照搬的点：
  - 典型聊天产品的消息流布局
  - 过重的机器人头像、对话气泡、语音助手感
  - 把入口做成大面积社交输入框
  - 强娱乐化的霓虹光效和夸张渐变
- 图片保存文件夹：`src/assets/references/chat-entry`

## 2. Balloon Motion

- 参考用途：寻找气球悬浮、绑线牵引、剪断上浮的动态气质参考，服务 Workbench 的核心交互记忆点。
- 推荐搜索关键词：
  - `floating balloon ui motion`
  - `soft hover motion design`
  - `object tether animation interface`
  - `cut rope interaction concept`
  - `lightweight floating object web animation`
- 适合借鉴的点：
  - 缓慢悬浮的节奏感
  - 轻微位移、旋转、呼吸感的控制方式
  - 物体被牵引时的张力表现
  - 点击后离场动画的方向感与轻盈感
  - 动效前后状态切换的清晰度
- 不要照搬的点：
  - 卡通过强、儿童玩具感过重的气球表现
  - 过于真实的 3D 派对气球质感
  - 复杂粒子、烟花、庆祝式反馈
  - 大幅抖动、弹跳过强、容易分散注意力的动效
- 图片保存文件夹：`src/assets/references/balloon-motion`

## 3. AI Workbench

- 参考用途：寻找整体 AI 工作台气质，包括桌面级信息组织、功能分区、留白和专业感。
- 推荐搜索关键词：
  - `ai productivity dashboard desktop`
  - `modern workbench interface`
  - `professional ai workspace ui`
  - `task orchestration dashboard`
  - `clean desktop productivity interface`
- 适合借鉴的点：
  - 专业但不冷硬的工作台氛围
  - 大块留白下的视觉秩序
  - 主操作区与辅助信息区的平衡
  - 清晰的页面层次和桌面画布感
  - 低饱和配色下的重点聚焦方式
- 不要照搬的点：
  - 赛博、黑客风、重科技炫技风
  - 普通数据大盘式密集布局
  - 纯聊天首页结构
  - 课程平台式分栏和学习进度表达
- 图片保存文件夹：`src/assets/references/ai-workbench`

## 4. Sidebar Layout

- 参考用途：寻找左侧导航的结构和气质参考，保证其像专业工具导航，而不是内容平台目录。
- 推荐搜索关键词：
  - `desktop app sidebar navigation`
  - `productivity software sidebar ui`
  - `clean collapsible sidebar`
  - `saas left navigation minimal`
  - `tool workspace navigation panel`
- 适合借鉴的点：
  - 导航层级的清晰表达
  - 折叠前后密度变化
  - 当前页面高亮方式
  - 图标、文字、间距的节制感
  - 与主内容区之间的边界处理
- 不要照搬的点：
  - 社区、课程、社交产品常见的内容流导航
  - 过于厚重或过度装饰的侧栏
  - 游戏化勋章、成长等级、数字激励入口
  - 把侧栏做成营销陈列区
- 图片保存文件夹：`src/assets/references/sidebar-layout`

## 5. Workflow Execute

- 参考用途：寻找工作流执行过程的参考，重点是步骤推进、信息补全、结果检查、版本输出的组织方式。
- 推荐搜索关键词：
  - `workflow modal ui`
  - `step by step task flow interface`
  - `guided form productivity app`
  - `review and refine workflow ui`
  - `structured ai workflow panel`
- 适合借鉴的点：
  - 多步骤流程的清晰分段
  - 当前步骤、后续步骤、完成状态的辨识方式
  - 表单、说明、预览区的组合关系
  - 质检提示与版本分支的可读性
  - 弹层或面板中的信息节奏
- 不要照搬的点：
  - 教学闯关、训练营、打卡任务式流程设计
  - 冗长的 onboarding 教程感
  - 过度复杂的流程图界面
  - 奖励驱动式完成反馈
- 图片保存文件夹：`src/assets/references/workflow-execute`

  - `usage insights panel productivity`
  - `professional notes dashboard`
  - `pattern library interface`
- 适合借鉴的点：

- 参考用途：寻找结果沉淀页的参考，重点看结果版本、复用入口、质检备注的组织方式。
- 推荐搜索关键词：
  - `result library ui`
  - `saved outputs dashboard`
  - `document version cards ui`
  - `content repository productivity app`
  - `review reuse results interface`
- 适合借鉴的点：
  - 列表与卡片混合的信息组织
  - 版本差异说明的展示方式
  - 结果摘要、用途说明、备注的层级

- 不要照搬的点：
  - 网盘式文件列表的机械感
  - 纯素材库或图片库视觉
  - 社交内容瀑布流
  - 游戏化收藏、勋章式展示
- 图片保存文件夹：`src/assets/references/result-library`

## 统一筛选标准

- 优先选择桌面端、专业工具、信息层次清楚的参考图。
- 优先选择低饱和、留白充分、结构稳定的界面。
- 如果某张图很美，但明显偏课程平台、聊天工具或游戏化产品，不建议纳入。
- 如果某张图局部细节可借鉴，可以保留，但要在使用时拆解成“布局”“气质”“交互提示”三个层面，而不是整体照搬。
